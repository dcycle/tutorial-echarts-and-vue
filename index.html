Resources

This project is aimed to demonstrate vue _without a build step_. In almost all cases, using the vue-cli will be preferred, however this can be a gentle introduction to vue.js; in some cases we hacked our way to making things work, such as in the "With input binding inside and outside of route" section (see also https://stackoverflow.com/questions/70625775/in-vue-3-how-to-make-v-model-work-within-a-router-template).
